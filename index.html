How to join the server

1. Get Modrinth

I'd usually recommend to use Curseforge since it's way more popular, has been available for ages now and whatnot but because Forge has been a pain in the ass during the setup of this modpack, I simply ditched it for Modrinth which is way faster, way more compact and way lighter than Curseforge (with a few minor downsides).


You can download Modrinth from here or get it directly here

Once downloaded, launch the executable and you should have Modrinth installed in no time !

2. Setup Modrinth

To actually launch and play Minecraft through Modrinth you will need to link your Mojang Account to Modrinth using the account tab located on the top-right of the app. Signing in will open a Microsoft login page in a new window. Simply log in and Modrinth will do the rest !

3. Get the modpack

Now that Modrinth is setup and ready to be used, get the file I've linked with my message (I'm linking it again just in case).


Now click on that '+' on the sidebar on the left side of the app to "Create new instance".

Then in the new window that appears, click on "From file" then "Import from file" and find that file I linked (the one called 'PervertedIntroverts 1.0.0.mrpack')

This should create a new instance, build it with Forge 47.3.29 (latest) for Minecraft 1.20.1, install all the mods you need to join and (if I got this right) you should also see the server in your server list upon launching the game.


Now all you gotta do is click that big 'Play' button and you're good to go !

3b. Just in case

If by some miracle I didn't get it right and the server doesn't appear in your server list, simply go to 'Multiplayer', click on 'Add server' and in the field that asks for an IP address paste this : character-represented.gl.joinmc.link


Then you should be able to join !
